# Fund Opportunity Case Study

A data analysis project focused on identifying and evaluating fund investment opportunities. The findings are visualized through an interactive Tableau dashboard designed for stakeholder review and decision-making.

> **Note:** The data provided in this repository is for testing purposes only and should not be used or distributed outside of approved channels.
